Пошаговое описание

Новая переменная, параметр:
Δ — минимально допустимое расстояние между последним из оставшихся дефолтных шагов и MaxBid.

1. Расчёт базовых шагов
Применяем дефолтный алгоритм.
Получаем исходный массив шагов
S = [s0, s1, …, sN]
2. Определение MaxBid
3. Проверка применимости нового алгоритма
max_bid < max(S)
Если false → вернуть исходный массив S и завершить работу.
Иначе → далее.
4. Построение промежуточного массива T
T = [ s ∈ S | s < MaxBid ]           
append MaxBid                       
5. Контроль расстояния Δ
Если в T единственный элемент (T = [MaxBid]) → T = [s0] и переход к шагу 6.
Иначе →
prev = T[ -2 ]                     // предпоследний элемент
if (MaxBid - prev) < Δ:
    удалить prev из T
6. Результат
Итоговый массив Result = T.
